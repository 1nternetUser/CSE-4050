What is JSON? (video version)

JSON stands for Java Script Object Notation.
It is a language independent text format that holds the following data types:

String
Number
Boolean Function
Array (ordered list of differing values) and
Object (unordered list of various values i.e. name types).

Since this is relatively easy to read for both humans and computers, it can be used to transfer data onto servers, webs, or mobile applications. It costs very little memory in comparison to XML.
